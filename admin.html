How it works

Your posts live in a Google Sheet.

1
You write here, or straight in the sheet
Publishing from this page adds a row to your sheet. You can also open the sheet and add or edit rows directly — same result.
2
The site reads the sheet
The Insights page loads your published rows every time a visitor arrives. New posts appear within seconds; no redeploy.
3
Unpublish any time
Set a row's "published" cell to FALSE (or delete the row) and the post disappears from the site.
One-time setup

Set up in about ten minutes.

Step 1 — Create the Google Sheet

Create a new Google Sheet, rename its tab to Posts, and put these headers in row 1, one per column:

slug   title   category   date   excerpt   body   published
Step 2 — Add the publishing script

In the sheet: Extensions → Apps Script. Delete any sample code and paste this, choosing your own secret key on the second line:

var SHEET_NAME = 'Posts';
var PUBLISH_KEY = 'choose-a-strong-secret-key';

function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    if (data.key !== PUBLISH_KEY) return json({ ok: false, error: 'Wrong publishing key' });
    var p = data.post || {};
    if (!p.title || !p.body) return json({ ok: false, error: 'Missing title or body' });
    var ss = SpreadsheetApp.getActiveSpreadsheet();
    var sh = ss.getSheetByName(SHEET_NAME) || ss.insertSheet(SHEET_NAME);
    if (sh.getLastRow() === 0) {
      sh.appendRow(['slug','title','category','date','excerpt','body','published']);
    }
    sh.appendRow([p.slug, p.title, p.category, p.date, p.excerpt, p.body, 'TRUE']);
    return json({ ok: true });
  } catch (err) {
    return json({ ok: false, error: String(err) });
  }
}

function json(o) {
  return ContentService.createTextOutput(JSON.stringify(o))
    .setMimeType(ContentService.MimeType.JSON);
}

Then: Deploy → New deployment → type Web app → execute as Me, who has access: Anyone → Deploy. Copy the web app URL into the "Publishing address" field on this page (it is remembered on this device), and use your secret key as the publishing key.

Step 3 — Connect the site to the sheet (once)

In the sheet: File → Share → Publish to web → choose the Posts tab and Comma-separated values (.csv) → Publish, and copy the link. Paste it between the quotes at the top of scripts/insights-feed.js:

window.SA_INSIGHTS_FEED_URL = "PASTE-THE-CSV-LINK-HERE";

Upload that one file (or redeploy once). This is the only redeploy you will ever need — every post after this goes live instantly.
